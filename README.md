# Next Marvel Movies

Este proyecto, como muestra el titulo nos brinda información sobre cual es la siguiente pelicula de Marvel, consumiendo una API externa y mostrando los datos necesarios por pantalla de forma amigable

## Tecnologias usadas

* PHP 
* Picocss
* Curl (consumo de API)
